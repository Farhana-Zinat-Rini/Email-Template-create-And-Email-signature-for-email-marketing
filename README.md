# 📧 Email Marketing & Email Signature Project

## Overview

This repository showcases a professional **Email Marketing** project developed as part of a **Digital Marketing** portfolio. The project demonstrates the creation of responsive email templates using **Mailchimp** and the design of a professional business email signature for consistent brand communication.

---

## Project Objectives

- Design a professional email signature
- Create responsive email templates
- Build marketing campaigns using Mailchimp
- Strengthen business branding through email communication
- Apply email marketing best practices

## Workflow

### Email Signature

1. Sign in to Gmail
2. Create a new email signature
3. Add profile information
4. Insert profile image
5. Add contact details
6. Include website and social media links
7. Save and test the signature

### Email Marketing

1. Log in to Mailchimp
2. Create an Audience
3. Create a Campaign
4. Select or design an email template
5. Add branding elements
6. Insert marketing content
7. Preview and test the email
8. Send the campaign
